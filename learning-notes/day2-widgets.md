# Day 2 - Flutter Widgets

Things I learned:
- Dart is very similar to C++ and C language
- Classes are important
- Everything in Flutter is a widget
- StatelessWidget = UI that does not change
- StatefulWidget = UI that can change

Widgets used:
- Text
- Center
- Column

Keypoints: 
- extension, super
- List<type> name;
- class name{}
- class name extends name{name(int something) : super (something)}
